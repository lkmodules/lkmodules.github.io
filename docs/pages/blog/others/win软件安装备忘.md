### [nvm-windows](https://github.com/coreybutler/nvm-windows)

``` shell
# nvm 版本
nvm version
# 下载指定 npm 版本
nvm install <version>
# 卸载
nvm uninstall <version>
# 使用
nvm use <version>
# npm 列表
nvm list
```
